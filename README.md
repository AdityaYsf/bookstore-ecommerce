# 📚 BookStore - Full Stack E-Commerce Application

Aplikasi toko buku online lengkap dengan fitur manajemen produk, keranjang belanja, dan dashboard admin.

## 🚀 Tech Stack

### Frontend
- **React 18** + Vite
- **Tailwind CSS** untuk styling
- **React Context API** untuk state management
- **React Router DOM** untuk navigasi
- **Axios** untuk HTTP client
- **Lucide React** untuk icons

### Backend
- **Node.js** + Express.js
- **PostgreSQL** database
- **Prisma ORM** untuk database management
- **JWT** untuk authentication
- **Bcrypt** untuk password hashing
